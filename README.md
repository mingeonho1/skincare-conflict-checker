# 같이써도돼? — 스킨케어 성분 충돌 체커

레티놀·비타민C·AHA/BHA 같은 액티브를 두세 개 들였는데 "같이 발라도 되나?"가 불안하다면, 전성분을 붙여넣기만 하면 30초 만에 답을 드려요.

## 핵심 기능

전성분을 최대 6개 제품까지 붙여넣으면 3가지 결과를 돌려드려요:

1. **충돌 경고** — 함께 쓰면 안 되는 성분 쌍을 메커니즘·심각도·대처법과 함께 알려드려요.
2. **안심 카드** — 경쟁 도구가 경고로 쏟아낼 때, 우리는 "함께 써도 괜찮은" 조합도 명시적으로 안심시켜 드려요. (나이아신아마이드+비타민C 위험설은 반박됐어요.)
3. **아침/저녁 사용 순서** — "충돌을 피하려면 어떻게 발라야 하나"를 광민감 여부·점도 순으로 정리해 드려요.

## 차별화

- **과잉 경고 없음**: 충돌이 없으면 초록 안심 메시지. 불필요한 공포 마케팅을 하지 않아요.
- **한국 제품 우선**: 전성분 한글 표기(아스코빅애씨드, 살리실릭애씨드 등)를 그대로 붙여넣어도 잡아요.
- **LLM 없이도 100% 작동**: 규칙 기반 엔진이 유일한 판정 진실의 원천이에요. Gemini API는 추출 정확도와 설명 자연스러움만 높여주는 부가 레이어예요.

## 기술 스택

- **Next.js 15** (App Router, Server Actions)
- **Zod v4** — 모든 입출력 경계 검증
- **Gemini 2.0 Flash** — 성분 추출 보조 + 결과 윤문 (optional, 폴백 완비)
- **Supabase** — 익명 성공 지표 계측 (`skinclash_check_log`)
- **Tailwind v4** + Pretendard — 신뢰감 있는 · 깨끗한 · 다정한 디자인

## 개발

```bash
pnpm dev        # 개발 서버
pnpm check      # lint + typecheck + test
pnpm build      # 프로덕션 빌드
```

필수 env:

```
SUPABASE_URL=
SUPABASE_PUBLISHABLE_KEY=
SUPABASE_SECRET_KEY=
GEMINI_API_KEY=   # optional — 없으면 키워드 폴백 모드로 동작
```
