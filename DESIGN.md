# DESIGN.md — 같이써도돼?

## Phase 0 — 무드보드

### 레퍼런스 & 훔칠 것

- **Toss**: 해요체 마이크로카피, 한 화면 한 메시지, 넉넉한 여백, 단일 Primary CTA. 결과 카드의 명확한 위계.
- **Stripe**: 결과 정보의 정렬과 밀도. 경고/상태별 색상 처리가 절제되어 있으면서도 즉각 인식됨.
- **Family(토스증권 아님)**: 부드러운 radius, 따뜻하고 깨끗한 빈 상태 일러스트 톤.

### 형용사 3개

**신뢰감 있는 · 깨끗한 · 다정한**

- 신뢰감: 의료/안전 정보 도구답게 과장 없이 사실적. 경고는 명확하되 무섭지 않게.
- 깨끗한: 정보가 많아도 숨 쉬는 여백. 불필요한 장식 0.
- 다정한: 입문자를 압박하지 않는 해요체, 초록 안심 카드, 빈 상태의 격려.

## Phase 1 — 팔레트·토큰

### 브랜드 컬러 근거

시맨틱 컬러(경고=레드, 안심=그린, 팁=옐로우)가 정보의 핵심 축이므로, 브랜드 컬러는
이 세 컬러와 충돌하지 않는 **차분한 슬레이트 블루-그레이** 계열로 선택.
"다 비슷한 그린 SaaS"와 구별되면서, 스킨케어 도구로서의 신뢰감(차갑지 않은 블루)을 전달.

- **브랜드**: `#4F7A9B` — 뮤트된 스틸 블루. 파랗지도, 회색도 아닌 신뢰감 있는 중간 톤.
- **브랜드-강**: `#3A6080` — hover/active용 10% 어두운 버전.

### 중립(ink) 스케일

브랜드 hue(블루-그레이 기조)를 살짝 섞어 순수 회색보다 고급스럽게:

- ink: `#1A2332` (블루기 도는 다크 네이비)
- ink-sub: `#4A5568` (슬레이트)
- ink-weak: `#8E99A8` (블루기 섞인 회색)
- bg: `#FAFBFC` (아주 살짝 쿨한 흰색)
- surface: `#F2F5F8` (카드 배경 — 차갑지 않은 쿨그레이)
- surface-raised: `#FFFFFF` (카드 자체, 배경보다 밝게)
- border: `#E2E8F0` (슬레이트 border)

### 시맨틱 컬러 (브랜드와 분리)

경고/안심/팁은 정보의 핵심 축이므로 브랜드 컬러와 완전히 독립:

- **warn-bg**: `#FFF5F5` / **warn-border**: `#FEB2B2` / **warn-text**: `#C53030`
- **warn-badge-bg**: `#FED7D7` / severity high: `#E53E3E` / moderate: `#DD6B20`
- **safe-bg**: `#F0FFF4` / **safe-border**: `#9AE6B4` / **safe-text**: `#276749`
- **tip-bg**: `#FFFBEB` / **tip-border**: `#FBD38D` / **tip-text**: `#92400E`

### 타이포 스케일

Pretendard Variable (기존 유지). 크기 점프 명확하게:

- xs: 12px / sm: 14px / base: 16px / lg: 18px / xl: 20px / 2xl: 24px / 3xl: 32px / 4xl: 44px

### Radius

부드러운(다정한) 쪽으로: 컨트롤 `10px`, 카드 `16px`, 태그/배지 `6px`.

### 간격 스케일

Tailwind 기본 4-scale 사용(4px 배수). 섹션 간 `py-16~24`, 연관 요소 `gap-2~4`.
