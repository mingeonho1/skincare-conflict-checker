# 레퍼런스 갤러리 — 무엇을, 어디서 훔칠 것인가

무드보드(Phase 0)에서 이 제품의 도메인·타겟에 맞는 2~3개를 고른다. **통째로 베끼지 말고, 한 가지씩 분명하게 훔친다.** 각 항목의 "훔칠 것"이 실제 디자인 결정으로 이어져야 한다.

## 도메인별 추천 조합

| 제품 성격                 | 1순위 레퍼런스           | 보조           |
| ------------------------- | ------------------------ | -------------- |
| 금융/계산/정산 도구       | Toss, Stripe             | Linear(절제)   |
| 개발자/B2B SaaS 도구      | Linear, Vercel, Raycast  | Stripe(밀도)   |
| 소비자 앱(생활/가족/습관) | Family, Airbnb, Duolingo | Toss(친근함)   |
| 콘텐츠/문서/노트          | Notion, Linear, Things   | Vercel(타이포) |
| 데이터 대시보드           | Stripe, Linear, Vercel   | Posthog        |
| 랜딩/마케팅 한 장         | Vercel, Stripe, Linear   | Arc            |

## 레퍼런스 카드

### Toss (toss.im)

- **훔칠 것**: 한 화면 한 메시지, 거대한 숫자 위계, 해요체 마이크로카피, 넉넉한 여백, 단 하나의 파란 CTA.
- **인상**: 신뢰감 · 친근 · 군더더기 없음. 한국 사용자 대상이면 카피·간격의 기준점.

### Linear (linear.app)

- **훔칠 것**: 절제된 모노크롬(거의 무채색) + 미묘한 한 포인트 컬러, 날카로운 작은 radius, 빠른 키보드 우선 인터랙션, 미세한 그림자/구분선, 다크 기반의 고급스러움.
- **인상**: 빠른 · 정밀 · 프로페셔널.

### Stripe (stripe.com)

- **훔칠 것**: 높은 정보 밀도를 정렬과 여백으로 정돈하는 법, 표·코드·숫자의 정렬, 절제된 보라/인디고 액센트, 신뢰를 주는 도큐먼트형 레이아웃.
- **인상**: 견고 · 정확 · 엔터프라이즈.

### Vercel (vercel.com)

- **훔칠 것**: 흑백 대비의 대담한 타이포, 모노스페이스 액센트, 극단적 여백, 미니멀 속 강한 헤드라인. 랜딩에 강하다.
- **인상**: 현대적 · 대담 · 기술적.

### Family (family.co) / Linear 류 소비자

- **훔칠 것**: 부드러운 radius, 풍부하지만 절제된 색, 촉각적인 마이크로 인터랙션, 둥근 카드, 따뜻한 빈 상태 일러스트.
- **인상**: 따뜻한 · 즐거운 · 손맛.

### Raycast (raycast.com)

- **훔칠 것**: 키보드 우선 UX, 명령 팔레트 패턴, 어두운 배경의 또렷한 위계, 아이콘 시스템.
- **인상**: 빠른 · 파워유저 · 깔끔.

### Notion / Things / Arc

- **Notion**: 콘텐츠 우선 타이포, 절제된 회색조, 블록 구조.
- **Things**: 여백과 타이포만으로 만든 고요함, 미세한 애니메이션.
- **Arc**: 대담한 컬러와 곡선, 유희적이되 정제된.

## 무드보드 작성 예시 (DESIGN.md 상단에 들어갈 형태)

```
## 디자인 아이덴티티
- 레퍼런스: Linear(절제된 모노크롬 + 한 포인트), Stripe(숫자/표 정렬)
- 훔칠 것: Linear의 거의-무채색 중립 스케일에 딥 그린 브랜드 1개만 / Stripe식 결과 표 정렬
- 형용사: 정밀한 · 차분한 · 신뢰감 있는
- 브랜드 컬러: #0B6E4F (딥 그린) — 금융 도메인의 안정감
- 중립: 그린 hue 살짝 섞은 슬레이트 그레이 스케일
- radius: 8px(날카롭고 정밀한 인상) / 폰트: Pretendard
```

## 판단 기준 (이 레퍼런스를 잘 썼는가)

- 완성된 화면을 처음 보는 사람이 "어, 이거 잘 만들었네"라고 느끼는가?
- 다른 팩토리 제품과 나란히 놓았을 때 **구별되는가?**
- 모든 색·간격·radius가 무드보드의 형용사 3개로 설명되는가?
